Made With {% mdicon heart, .inline %} By Good Robotics

Made possible by Crowd Supply and sponsored by You, our backers. Thank you.

We Love Makers. This is a pre-release board meant to demonstrate how a DIYer could control a stepper motor using WiFi. We want to make it as easy as possible to embed a stepper motor into a professional first class project. We hope you enjoy using this board! Any feedback would be greatly appreciated: aklofas@goodrobotics.net
