{% mdicon wrench %}Hookup guide

This board is a WiFi module connected to stepper motor driver. It combines both the driver control logic (such as velocity commands, absolute position commands, etc.) with the current/voltage H-Bridge controller that drives a single stepper motor. To get started, you'll need:

  1. 12V to 80V DC voltage source. You can use a standard laptop power brick with a barrel connector (5.5mm OD x 2.5mm ID center positive) or any power supply that can drive enough current (you probably want at least 2-3 Amps).
  2. Bipolar Stepper Motor. There are many different kinds. For beginners we recommend a NEMA 17 because it's a good tradeoff between size and power. If you use a unipolar stepper motor you'll need to wire the leads so that it is electrically identical to a bipolar motor.

For convienence, we've added a barrel connector jack. You can use either the screw terminals or barrel jack for voltage input (they are connected in parallel).

There will be four wires (two pairs of wires) from your stepper motor that you'll need to connect to the board. They'll most likely be all different colors, so determining which are the pairs is important. The easiest way is to look at the datasheet that came with your stepper motor. If you don't want to do that, connect an ohm meter to two pairs and test for connectivity. Pairs could read anywhere fom 0 to 10 ohms. Repeat the test using all combinations of wires until you've found the pairs. Both pairs A & B will be wired to the board as shown in the following figure.

Polarity is not important because the software has a Reverse Wires configuration setting that will reverse the direction in software. (Reverse polarity = reverse direction). If you don't want to use that configuration setting and the motor is spinning backwards, choose either A or B wires (but not both), and reverse the (+) plus and (-) minus colors.

{% mdicon flash %}High Voltage Operation (50 volts and above)

With proper configuration, this board can handle input voltages up to 80 volts. However, board modification is needed when driving 50 volts and higher. This is due to two factors: 1) Input ratings on the barrel connector 2) the maximum input voltage of the DC-DC regulator. When operating at 50+ volts, please ensure the following:

  1. Use only the screw terminals for voltage input. (The barrel connector is not rated for more than 50 volts)
  2. Cut the jumber JP0 (located mid-board next to DC-DC regulator) and provide 3.3 volt logic supply seperately to one of the breakout pins labeled 3V3 and GND on either side of the board.
{% mdicon bomb %}Hardware Reset

There is a hard reset button located on the board that will restore the configuration to factory defaults. The hard reset procedure is as follows:

  1. With the board not powered, press and hold the reset button.
  2. Power the board with the button still pressed and hold for 3 seconds.
  3. Release the button. The board will now wipe the configuration files and revert to factory defaults.
